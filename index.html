Amazeballs

About

The path is generated by a depth-first recursion algorithm, as described by Mike Bostock, implemented in Angular.js. The maze itself is simply an HTML table element with chunky borders. The underlying model is a two-dimensional array.

Javascript's native handling of arrays is a little unusual:

            [1, 2] + [3, 4]
            #=> "1,23,4"

Fortunately Sylvester provides a nice implementation of vectors, saving a lot of awkward, repetitive code.
